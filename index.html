Two things: Return the organizationName from the AddressBook. And escape any ' in the addresses. OCABMutableMultiValue is processed by SBJsonWriter which does escape " amongst other characters but not ' Put in a contact with an organization/company name and a line in the address like "87 Queen's Street", click the button and it will process the returned data correctly which will also include the organization name. Previously it would choke on the data at the key/value when a value looks like '87 Queen's Street'.
